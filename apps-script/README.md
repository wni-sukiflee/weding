# วิธีเชื่อมปุ่มตอบรับ (RSVP) เข้ากับ Google Sheet + กราฟ

เมื่อแขกกด **"ตอบรับด้วยความยินดี"** หรือ **"ขออภัย ไม่สามารถมาได้"** บนการ์ด
ข้อมูลจะถูกบันทึกลงชีต `งานแต่งงาน 26 ก.ค` และมีกราฟเทียบจำนวน "มาได้ vs มาไม่ได้" ให้อัตโนมัติ

ทำตามนี้ **ครั้งเดียว** (ประมาณ 5 นาที):

---

## ขั้นที่ 1 — วาง Apps Script ลงในชีต

1. เปิดชีต **งานแต่งงาน 26 ก.ค** (ไฟล์ที่ส่งมา)
2. เมนูบน → **ส่วนขยาย (Extensions)** → **Apps Script**
3. ลบโค้ดเดิมที่อยู่ในไฟล์ `Code.gs` ทิ้งทั้งหมด
4. เปิดไฟล์ [`apps-script/Code.gs`](./Code.gs) ในโปรเจกต์นี้ → คัดลอกทั้งหมด → วางลงไป
5. กดรูป 💾 (บันทึก)

## ขั้นที่ 2 — สร้างหัวตาราง + กราฟ

1. ในหน้า Apps Script ด้านบน เลือกฟังก์ชัน **`setupSheet`** จาก dropdown
2. กด **▶ Run**
3. ครั้งแรกจะมีหน้าต่างขออนุญาต → **Review permissions** → เลือกบัญชี Google ของคุณ
   → ถ้าขึ้น "Google hasn't verified this app" ให้กด **Advanced** → **Go to (ชื่อโปรเจกต์) (unsafe)** → **Allow**
   *(ปลอดภัย เพราะเป็นสคริปต์ของคุณเอง)*
4. กลับไปดูที่ชีต จะมีแท็บใหม่ **"ตอบรับ"** (เก็บข้อมูล) และ **"สรุป"** (มีกราฟวงกลม) เพิ่มขึ้นมา

## ขั้นที่ 3 — Deploy เป็น Web app

1. ในหน้า Apps Script มุมขวาบน → **Deploy** → **New deployment**
2. กดรูปเฟือง ⚙ ข้าง "Select type" → เลือก **Web app**
3. ตั้งค่า:
   - **Description:** อะไรก็ได้ เช่น `rsvp`
   - **Execute as:** **Me (อีเมลของคุณ)**
   - **Who has access:** **Anyone** ← สำคัญมาก (ต้องเป็น Anyone ไม่ใช่ Anyone with Google account)
4. กด **Deploy** → อนุญาตอีกครั้งถ้าถูกถาม
5. คัดลอก **Web app URL** (ลงท้ายด้วย `/exec`)

## ขั้นที่ 4 — ใส่ URL ลงในเว็บ

1. เปิดไฟล์ [`site.jsx`](../site.jsx) (อยู่บนสุดของไฟล์)
2. หาบรรทัด:
   ```js
   const RSVP_ENDPOINT = "";
   ```
3. วาง URL ที่คัดลอกมาไว้ในเครื่องหมายคำพูด:
   ```js
   const RSVP_ENDPOINT = "https://script.google.com/macros/s/AKfyc.../exec";
   ```
4. บันทึก แล้ว deploy เว็บใหม่ (push ขึ้น Vercel/ที่โฮสต์ไว้)

> 💡 บอกผม (Claude) ว่า URL คืออะไร เดี๋ยวผมใส่ให้ + bump cache + push ให้เลยก็ได้

---

## ตรวจสอบว่าใช้งานได้

- เปิด Web app URL ตรง ๆ ในเบราว์เซอร์ ควรเห็น `{"ok":true,"comingGuests":0,"notComing":0}`
- ลองกดตอบรับบนการ์ดจริง 1 ครั้ง → แถวใหม่ควรโผล่ในแท็บ "ตอบรับ" และตัวเลข/กราฟในแท็บ "สรุป" อัปเดต

## ข้อมูลที่เก็บ
| คอลัมน์ | ความหมาย |
|---|---|
| เวลา | เวลาที่กดตอบรับ |
| ชื่อ | ชื่อที่แขกกรอก (ถ้ากรอก) |
| สถานะ | `มาได้` หรือ `มาไม่ได้` |
| จำนวนท่าน | จำนวนคนที่จะมาด้วย (ค่าเริ่มต้น 1) |

กราฟวงกลมในแท็บ "สรุป" แสดง **สัดส่วน %** ของคนที่มาได้ vs มาไม่ได้ และอัปเดตเองทุกครั้งที่มีคนตอบ

## อัปเดตโค้ดภายหลัง
ถ้าแก้ `Code.gs` แล้ว ต้อง **Deploy → Manage deployments → ✏️ แก้ deployment เดิม → Version: New version → Deploy**
(ถ้า New deployment ใหม่ URL จะเปลี่ยน ต้องไปแก้ใน `site.jsx` อีก)
